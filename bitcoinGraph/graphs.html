Line Chart using D3.js

This graph is generated using the d3.js library. The data is the Bitcoin price index from the past month as of today. Javascript explanation: Once the DOM has loaded, the data from the API is fetched and then parsed into an array. Depending on whether the promise from the API is successful or not, the .then () function will perform subsequent actions to fetch the data.
