This is a breakout game that I created using HTML canvas and Javascript. The canvas refreshes rapidly to produce a smooth animation. After each refresh, the position of the ball is compared to the position of the bricks. The bricks are created using an array that specifies whether it has been hit or not. If the ball hits one of the bricks, it's status changes so it is no longer drawn and no longer detected by the ball. To the user the brick just disappears.
