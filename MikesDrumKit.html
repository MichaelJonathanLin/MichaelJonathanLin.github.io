I created a drumkit by following day 1 of the Javascript 30 challenge. This page involved using addEventListener so that the Javascript could respond to the HTML events. In particular, when a key is pressed, an html audio element is triggered, as well as a key animation. When the css animation transition ends the keys are reset back to normal.
